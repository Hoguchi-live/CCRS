int apply_key ( MG_curve_t *  rop,
MG_curve_t *  op,
key__t *  key,

Walk in the isogeny graph starting from op following the path given by the key. The curve rop is set to the endpoint of this walk. rop must be initialized. The l-primes are ordered according to their working extension. That way we just change the extension degree when needed. Returns 1 if successful and 0 if an error occured during a walk.
